# Software Process Model

در این دایرکتوری چرایی و پردازشِ مدلِ نرم‌افزار (Software Process Model) و چرخهٔ حیات محصول (Product Life Cycle) مدیا بررسی می‌شود. تغییرات و پیشرفت مدل و چرخهٔ حیات آن نشان داده می‌شود، دلایل ذکر می‌شود، مراحلِ توسعه کنترل می‌شود، نیازمندی‌ها ذکر و لیست خواهد شد، و نتایج به نمایش گذاشته می‌شود.
